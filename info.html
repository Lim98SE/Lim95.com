L-DOS Help Service

Command List

echo - Prints to the command line

EXAMPLE: echo hello

RESULT: : hello

break - closes L-DOS

dir - views directory

dirs [term] - searches directory

findstr [term] [file] - searches for a string in a file

md [name] or mkdir [name] - creates directory

cd [name] - changes directory

del [file] - deletes file

copy [file] - clones a file

cls or command - clears screen

backup - writes a backup to bk.py

chkdsk - shows free, used, and total space on current drive

pd - previous directory

new [name]- creates a new file, then opens it in the default Windows app

ping [ip] - pings an ip adress at HTTP and HTTPS

help - shows this page
