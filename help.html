Mouse Helper GDevelop Extension

Functions

1. Is Clicked?

Easy way to check if an object is clicked.


2. Go To Mouse

Brings an object to the mouse.


3. Is Clicking

Checks if the mouse is being clicked.


4. Is Clicking Cooldown

Only checks once.


5. Follow Mouse

Uses the TWEEN behaviour to make an object follow the mouse.


6. Mouse Cursor

Allows you to make a custom cursor for your game.
