L-DOS

What is L-DOS?

L-DOS is my Python text-based shell.

You can run commands, read files, and in the

first version, use a graphical interface.

So what does L-DOS even do? Why should I want it?

Well, L-DOS can do many things, such as:

  • Bring up "nostaliga" from the MS-DOS days
  • Read a ton of file formats as well as play .wav files
  • Have a GUI (only in L-DOS 0.1.x and L-DOS X)
  • Save some configuration stuff
  • And maybe some more stuff!

    I downloaded L-DOS, how do I use it?

    L-DOS was written in Python 3.6.2, but it may be backwards compatible with lider versions.

    If you do any testing, email me results at lim95.yt@gmail.com. Until then, assume

    that L-DOS 0.1.x and 2.0 will only work with anything above 3.6.

    That doesn't answer my question- how do I run L-DOS?

    You can't do it just by running from the conslie, or double-clicking the file. It isn'r a module either.

    To run L-DOS, you have to do the following:

  • Open it in the IDLE editor
  • Click "run in python shell
  • Type "boot()" into the prompt
  • Make sure you're in a directory you can edit in
  • Enjoy L-DOS
